# AI-Assisted DIF Hypothesis Generation Research Workspace

## Working Idea

Can large language models generate prospective, text-based hypotheses about
which psychological scale items will show differential item functioning (DIF),
and can those hypotheses be validated against empirical psychometric models such
as longitudinal ordinal MNLFA?

## Bottom Line

This is not strong enough as a post-hoc "LLM explains our MAPS DIF results"
paper. It becomes publishable only if framed as a preregistered, blinded,
robustness-heavy hypothesis-generation study.

The current MAPS acculturative-stress item set is too small by itself. With 8
items and 4 DIF covariates, there are only 32 item-by-covariate prediction units.
For a standalone paper, add more items, a second instrument, or an external DIF
dataset.
